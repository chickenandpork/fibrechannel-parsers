-N|–nickname={file/uri} Import nicknames by parsing a text stream from various sources

java -jar fcparsers.jar --nickname=switch44.zoneshow
Parse results for AliShowZoneParser:
Zones: 44
Aliases: 112 (names with one or more WWPNs)
Aliases: 136 (name/WWPN tuples)

In this example, a zone file was parsed by the AliShowZoneParser resulting in 112 nicknames; due to duplicate nicknames, there are actually 136 unique WWPN/alias tuples, which means that (136-112) 24 of the WWPNs have the same alias as other WWPNs
